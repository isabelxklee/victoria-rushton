# Victoria Rushton Typography Portfolio ✒️ 

## Getting started

1. Clone this repo and cd into the directory.

2. Install all dependencies.

```
$ npm install
```

3. Start Gatsby server.

```
$ npm start
```

4. Navigate to `http://localhost:8000`.
